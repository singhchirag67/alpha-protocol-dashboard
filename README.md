# alpha-protocol-dashboard

Personal Evolution Dashboard
A self-built, interactive dashboard designed to optimize physical growth, mental discipline, and academic performance.
This project combines science-based fitness, nutrition planning, and discipline frameworks into a single, accessible interface.

🎯 Features
📊 Growth Tracking Dashboard (weight, height, projections)
🌙 Sleep Optimization Module (HGH-focused insights)
🥗 Sattvic Nutrition Plan for clean bulking
⏱️ 15-Min Workout Protocol (high efficiency training)
🧘 Mindset System (Brahmacharya + Focus)
📈 Interactive charts using Chart.js
🎨 Clean UI built with Tailwind CSS

🛠️ Tech Stack
HTML5
Tailwind CSS
JavaScript
Chart.js
📱 Accessibility

This is a single-file web app (AlphaProtocol.html)
Works on:
💻 Laptop
📱 Mobile
📟 Tablet

Just open in any browser—no installation needed.

🎯 Purpose
Built as a personal self-improvement system to:
Improve physique
Maximize height potential
Build discipline
Stay focused for Class 10 Board 2027
